WannaCry病毒全球爆发，新华三绿洲平台建议您做好安全防范措施

一、事件综述

目前，全球各地成千上万台系统，包括美国、英国的众多电脑和服务器遭遇了WannaCry勒索病毒的袭击。在国内，高校和医院则成了重灾区。这种勒索病毒名为WannaCry，从部分学校遭遇的大量感染事件上看，大量论文等重要文件被加密，只有使用比特币支付赎金才能恢复。

此病毒的传播手段利用了4月14日黑客组织ShadowBrokers公布的EquationGroup黑客工具，其中ETERNALBLUE模块是SMB 漏洞利用程序，可以攻击开放了 445 端口的 Windows 机器，实现远程命令执行。

截至发稿，由该勒索软件导致的攻击事件已经在全国乃至全球造成非常严重的危害，不少企业内网也已经面临此类威胁，目前对该勒索软件加密的文件无法解密，请广大客户注意防范。 当系统被该勒索软件入侵后，弹出勒索对话框：

传播原理

该勒索软件迅速感染全球大量主机的原因是利用了基于445端口传播扩散的SMB漏洞MS17-010，微软在今年3月份发布了该漏洞的补丁，请尽快修复。

Wannacry 细节描述

由于以前国内多次爆发利用445端口传播的蠕虫，部分运营商在主干网络上封禁了445端口，但是教育、医疗等网络并没有做此限制，仍然存在大量暴露445端口且存在漏洞的电脑，导致目前蠕虫的泛滥。

二、无线局域网解决方案

新华三无线设备用户可以采取以下措施，阻断病毒的传播途径，避免爆发式感染。这些措施任意一项都可以起到阻断病毒传播的效果，采取措施前请慎重选择适合您使用场景的方案。

1 开启用户隔离

通过隔离无线局域网中的终端设备来阻断病毒传播
适用于终端之间没有互相访问需求的场景
适用于H3C Comware V7无线控制器 集中/本地转发模式
适用于H3C Comware V5无线控制器 集中转发模式

AC配置示例：
#
wlan service-template H3C
  ssid H3C-WiFi
  vlan 1000
  user-isolation enable
  service-template enable
#

注意：
  • 部分设备启用改命令行时，需要先关闭无线服务模板，会造成临时业务中断
  • 采取此项措施后，无线局域网的用户将不能相互访问，可能带来诸如不能使用FTP服务器等后果

2 启用ACL封杀病毒报文传播端口

通过封杀病毒传播使用端口来阻断病毒传播
适用于开启无线认证的场景
适用于H3C Comware V7无线控制器 集中/本地转发模式
适用于H3C Comware V5无线控制器 集中/本地转发模式

AC配置示例：
#
domain cloud
  authorization-attribute acl 3333
  authentication portal none
  authorization portal none
  accounting portal none
#

#
acl advanced 3333
  rule 0 deny tcp destination-port eq 135
  rule 1 deny tcp destination-port eq 137
  rule 2 deny tcp destination-port eq 445
  rule 3 deny tcp destination-port eq 139
  rule 4 deny udp destination-port eq 135
  rule 5 deny udp destination-port eq netbios-ns
  rule 6 deny udp destination-port eq 445
  rule 7 deny udp destination-port eq netbios-ssn
  rule 8 permit 0
 #

注意：
  • 仅对已经开启了无线认证的场景有效
  • 3333为示例ACL编号，用户可根据实际情况编号

您可以通过绿洲“管理 - 运营维护 - 命令助手”功能来下发配置，也可以通过登录设备控制台来下发这些命令。

三、其他防范措施

1、边界防护

强烈建议网络管理员在网络边界设备、防火墙上阻断对135/137/139/445端口的访问；

如果边界上有IPS、NGFW等防护设备，请升级设备的IPS、AV规则到最新版本，直到确认网内的电脑已经安装了MS07-010补丁或关闭了Server服务。

2、系统加固

及时升级操作系统到最新版本；

对重要文件进行非本地备份；

停止使用Windows XP、Windows 2003等微软已不再提供安全更新的操作系统；

通过windows防火墙高级配置，对445端口的访问进行限制。

3、根治手段

目前微软已发布补丁MS17-010修复了此漏洞，请立即安装。

4、虚拟补丁

目前新华三IPS 特征库已经可以对MS17-010进行防御，建议将特征库升级到最新版本并开启下列规则进行防护并配置为阻断。新华三安全攻防团队正在对已经捕获的病毒样本进行分析，后续将发布防护病毒规则，以及关于该病毒进一步的分析报告，敬请关注。
